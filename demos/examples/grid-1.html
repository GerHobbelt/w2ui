Simple Grid

The grid object below displays data loaded from a url in JSON format. As you see the grid is extremely simple to use and easy to enable more features and listen to events.

You can define the size of the columns in % or px or both at the same time. When % and px are defined, it will first assign width to the columns defined in px, then it will calculate the size of the column in % based on the remaining px.
$(function () { $('#grid').w2grid({ name: 'grid', url: 'data/list.json', columns: [ { field: 'fname', caption: 'First Name', size: '30%' }, { field: 'lname', caption: 'Last Name', size: '30%' }, { field: 'email', caption: 'Email', size: '40%' }, { field: 'sdate', caption: 'Start Date', size: '120px' } ] }); });
