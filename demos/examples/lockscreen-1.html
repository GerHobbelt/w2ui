LockScreen

Locks / unlocks the screen: click the button to lock the screen (a grey overlay is shown) and click anywhere in the overlay to unlock the screen again; these are internal methods provided by the w2popup module.
function lockScreen() { w2utils.lock('body', { msg: 'Locking the screen...', opacity: 0.5, spinner: true }); var lock = $('.w2ui-lock, .w2ui-lock-msg'); lock.on('click', function (e) { w2utils.unlock('body'); }); }
