Resets the grid to the initial state.
reset()

Description

The .reset() method will not touch records array, but will reset the following elements of the grid: search fields, sort fields, no records will be selected, hidden records will be displayed, first page will be opened, vertical scroller will be moved to top, .isLoaded property will be set to false and .refresh() method will be called.
