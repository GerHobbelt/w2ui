Loads new locale
w2utils.locale(locale);
locale string or path, name of the locale or path to the locale file

Description

All locale files are in the /src/locale folder (see any one as an example). The first parameter can be a string or a path the the file. If it is a string (ex: ru-ru) it is the name of the locale. The method will perform sunchronous load of the locale, so it will pause until locale is loaded.
